# 🔐 Security Auth Core
> "Robust Identity Management & Enterprise Data Encryption."

이 리포지토리는 금융 및 엔터프라이즈 급 보안 표준을 준수하는 **Authentication & Authorization** 기술의 집약체입니다.

## 🛡️ Security Architecture
### 1. Advanced Authentication
- **OAuth2 & JWT**: 무상태성 인증 시스템 및 토큰 보안 강화.
- **RTR (Refresh Token Rotation)**: 토큰 탈취 방지를 위한 주기적 갱신 전략.

### 2. Enterprise Encryption
- **Hybrid Encryption**: RSA(Key Exchange) + AES-256(Data Encryption).
- **KMS Integration**: 하드웨어 보안 모듈 및 키 관리 시스템 연동 전략.
